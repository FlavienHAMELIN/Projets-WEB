Accéder à notre site :
1) Placer le dossier FlayukiShop dans votre dossier www

2) Dans PHPMyAdmin, créer une nouvelle base de données nommée site-e-commerce

3) Importer la BDD site-e-commerce.sql situé dans le dossier FlayukiShop depuis la base que vous venez de créer:
	->Cliquer sur site-e-commerce ->Importer ->Sélectionner le site-e-commerce.sql -> Exécuter

4) Depuis Uwamp ou WampServer, accéder au navigateur www 

5) Cliquer sur FlayukiShop

6) Bienvenue sur notre site !


Accéder à l'espace administrateur:
1) Accédez à l'url http:localhost/FlayukiShop/admin/index.php

2) Connectez-vous: ID = admin et Password = flayuki

3) Administrer. 

NB: 
- Pour écrire un apostrophe: \'
- Les flottants s'écrivent avec un point, pas une virgule.
- Pour lier une image d'un produit à ajouter, il faut au préalable la placer dans le dossier admin.
